# 🏯 思考道場

ローカルLLM（Ollama）を使って「頭を良くする」ためのトレーニングアプリです。  
APIキー不要・完全無料・全データはブラウザのlocalStorageに保存されます。

## スクリーンショット

> ※ スクリーンショットは後日追加予定

## 機能

### 5つのトレーニングモード

| モード | 内容 |
|--------|------|
| 🧘 対話 | 問いを重ねて思考を深める。禅問答も選択可 |
| ⚔️ 反論トレーニング | AIがあえて反論してくる。攻守切替スイッチあり |
| 🧭 意思決定ログ | 日常の決断をAIと構造化し、言語化する習慣を身につける |
| 📰 アウトプット力 | 文章を要約・説明してAIに採点してもらう |
| 🌍 国際情勢・政治・経済 | Reuters・NHKから最新ニュースを自動取得して分析 → 確認クイズ |

### その他の機能

- **AIコーチのカスタマイズ** — 名前・職業・よく悩むテーマ・コーチスタイル（4種）を登録
- **語彙強化** — ビジネス用語・新聞/法律レベルの語彙（計50語）を対話中に自然に習得
- **デイリータスク & ストリーク** — 1往復12pt、100ptでデイリー達成。連続日数を🔥表示
- **カレンダー** — 達成した日に✓がつく
- **思考レベルスコア** — 5往復後に深さ・語彙・論理を10点満点で自動採点
- **バトルスコアバー** — 反論モードでAI vs ユーザーのスコアをリアルタイム表示

## 必要なもの

- [Ollama](https://ollama.com/) がインストール・起動済みであること
- Python 3系（ローカルサーバー用）
- `qwen2.5:7b` モデル（推奨）

## セットアップ

### 1. Ollamaのインストール

[https://ollama.com/](https://ollama.com/) からインストール。

### 2. モデルのダウンロード

```bash
ollama pull qwen2.5:7b
```

### 3. CORS設定（初回のみ）

Ollamaがブラウザからのリクエストを受け付けるよう設定します。

**Windowsの場合：**
1. スタートメニュー → 「環境変数」を検索 → システム環境変数の編集
2. 「新規」をクリック
3. 変数名：`OLLAMA_ORIGINS`、変数値：`*` を入力してOK
4. Ollamaを再起動

### 4. アプリの起動

```bash
git clone https://github.com/aliks-kuro/thinking-dojo.git
cd thinking-dojo
```

`start.bat` をダブルクリックするとブラウザが自動で開きます。

または手動で：

```bash
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

## 使い方

1. **⚙️ 設定** からプロフィール（名前・職業・コーチスタイル）を登録
2. ホーム画面からトレーニングモードを選択
3. 毎日100ptを目指してストリークを継続

## 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（シングルファイル、フレームワーク不使用）
- **LLM**: [Ollama](https://ollama.com/) + `qwen2.5:7b`
- **ニュース取得**: RSS（Reuters・NHK・Google News）+ [rss2json.com](https://rss2json.com/)
- **データ保存**: ブラウザの localStorage
